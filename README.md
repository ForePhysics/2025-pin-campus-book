# For developers

1.  **首先安装依赖**

    在项目根目录下运行以下命令以安装所有依赖：

    ```shell
    npm install
    ```


2.  **启动开发服务器**

    使用以下命令启动本地开发服务器：

    ```shell
    npm run develop
    ```

    服务器启动后，默认会在 [http://localhost:8000](http://localhost:8000) 运行。
